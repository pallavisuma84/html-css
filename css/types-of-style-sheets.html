Inline style sheet

"This is inline stylesheet where we are mentioning all the required stles to the particular element.Using this is not a good practice,because we have to mention style to every element it become tough.This inline style sheet is used to Quick check the styles"

External style sheet

"In this external style sheet we mention the styles in an external file and then link that file with the help of link rel and href elements.This is good to use when we want to maintain same styles throughout all the web pages in web site "
